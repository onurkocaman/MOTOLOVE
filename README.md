MotoLove - Motosiklet İhale Sistemi

MotoLove, motosiklet alım-satım süreçlerini ve ihale mekanizmasını yönetmek amacıyla geliştirilmiş web tabanlı bir uygulamadır.

Temel Fonksiyonlar

İhale Sistemi: Kullanıcıların yayındaki ilanlara teklif verebilmesi ve en yüksek teklif takibi.

İlan Yönetimi: Kullanıcı arayüzü üzerinden detaylı ilan oluşturma ve listeleme.

Yönetim Paneli: Admin yetkisiyle ilan onaylama, kullanıcı listeleme ve sipariş kontrolü.

Teknik Altyapı

Backend: Python (Flask)

Veritabanı: SQL (İlişkisel modelleme)

Frontend: JavaScript, HTML, CSS (Bootstrap)

Kurulum

pip install flask

python app.py komutu ile yerel sunucuyu başlatın.

Tarayıcıdan localhost:5000 üzerinden erişim sağlayın.
